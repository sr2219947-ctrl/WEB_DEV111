What are HTML Entities?

HTML entities are special codes used to display reserved characters and symbols in webpages.

Entities usually begin with an ampersand (&) and end with a semicolon (;).

Reserved Character Entities

Symbol Entity Name Description
< &lt; Less Than
> &gt; Greater Than
& &amp; Ampersand
" &quot; Double Quotes
' &apos; Single Quote

Common Symbol Entities

Symbol Entity Name Description
© &copy; Copyright Symbol
® &reg; Registered Trademark
™ &trade; Trademark Symbol
♥ &hearts; Heart Symbol
☆ &star; Star Symbol
